## Step 8.1 结论 — 多周期趋势过滤（方案A）❌ 无效

**实验内容**：日线 EMA20/EMA60 趋势过滤，DOWN 时阻止买入，SIDE 时仅保留 STRONG_BUY。

**关键发现**：
- 训练集平均收益：基线 -1.16% → v8.1 **-2.33%**（恶化 1.17 个百分点）
- 验证集平均收益：基线 +0.40% → v8.1 **+0.16%**（恶化 0.24 个百分点）
- 训练集 0/10 只股票改善，验证集仅 4/10 只改善

**失败根因 — 级联时间线偏移效应**：
- 硬过滤器阻止买入 → 改变首次入场时机 → 后续交易链完全偏移
- 推迟入场的价格可能更差 → 更频繁触发止损 → 反而交易次数增加
- 部分股票（中国平安、长江电力）v8.1 交易次数反而大幅增加

**结论**：日线趋势硬过滤不适用于当前策略框架，放弃此方案。

## Step 8 最终结论 — v8 正式版（8.3a 配置）

**v8 推荐参数**：
```
# 继承 v7.2 基础参数
dragon_lookback:        5
stop_loss:              -5%（ATR 关闭时的后备）
trailing_activate:      +5%（ATR 关闭时的后备）
trailing_stop_pct:      3%（ATR 关闭时的后备）
time_stop_bars:         40
max_consecutive_losses: 3

# v8 新增（8.3a）
use_atr_stops:          True
atr_stop_multiplier:    2.0    # 止损距离 = 2×ATR / 买入价
atr_tp_activate:        1.5    # 移动止盈激活 = 1.5×ATR
atr_trail_multiplier:   1.0    # 移动止盈回撤 = 1.0×ATR
atr_stop_floor:         -3%    # 止损下限（最窄）
atr_stop_ceil:          -8%    # 止损上限（最宽）
use_vol_filter:         True   # 量能仓位系数开启
```

**v8 vs v7.2 关键数据（10 只股票）**：

| 指标 | v7.2 基线 | v8（8.3a） | 变化 |
|------|-----------|-----------|------|
| 训练集平均收益 | -1.16% | **-0.95%** | +0.21% |
| 验证集平均收益 | +0.40% | **+0.46%** | +0.06% |
| 验证集逐股胜率 | - | **8/10** | |
| 验证集盈亏比 | 1.31 | **1.75** | **+34%** |
| 验证集最大回撤 | -1.83% | **-1.52%** | **-17%** |
| 训练集盈亏比 | 0.52 | **0.89** | **+71%** |

**第八阶段核心认知**：
1. **硬过滤有害**：阻止信号（WAIT_CONFIRM=0%、日线趋势阻止买入）会产生级联时间线偏移
2. **软过滤有效**：量能仓位系数（放量加仓/缩量减仓）改善风险收益比而不破坏交易序列
3. **ATR 止损改善有限但方向正确**：单独效果中性，但作为量能过滤的基座提供了波动率感知
4. **边际效益递减**：叠加过多优化反而互相干扰，简洁组合优于复杂堆叠
