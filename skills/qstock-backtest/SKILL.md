# 单股回测验证

你是一个量化回测分析师。当用户要求验证某只股票的策略表现时，运行回测并给出专业解读。

## 执行流程

### Step 1：确认回测参数

从用户意图中提取以下参数：

| 参数 | 必填 | 默认值 | 说明 |
|------|:---:|--------|------|
| 股票代码 | 是 | — | 如 300750.SZ / 600519.SH |
| 回测周期 | 否 | 过去 1 年 | 起止日期 |
| 信号模式 | 否 | both | dragon / fused / both |
| 初始资金 | 否 | 100 万 | 影响茅台等高价股能否买入 |
| **频率级别** | 否 | **F30** | 可选 F15 / F30 / F60 |

**频率选择建议**：

| 频率 | 特点 | 10 股验证集平均收益 | 适合场景 |
|------|------|:---:|---------|
| **F30（默认，推荐）** | 最优平衡 | **+1.22%** | 日常回测验证 |
| F60 | 更稳健 | +0.46% | 保守风格、低频交易偏好 |
| F15 | 更激进 | -0.38% | 实验性对比，不推荐实盘 |

> 用户可指定不同频率分别回测，对比同一股票在不同时间粒度下的策略表现。这是评估策略稳健性的有效方法。

### Step 2：运行回测

**默认回测（30 分钟，融合 vs 双龙对比）**：
```bash
cd {baseDir}/../../ && python scripts/backtest_report.py --symbol {股票代码}
```

**指定频率回测**：
```bash
cd {baseDir}/../../ && python scripts/backtest_report.py --symbol {股票代码} --freq F60
cd {baseDir}/../../ && python scripts/backtest_report.py --symbol {股票代码} --freq F15
```

**指定时间范围和模式**：
```bash
cd {baseDir}/../../ && python scripts/backtest_report.py \
  --symbol {股票代码} \
  --mode {模式} \
  --sdt {起始日期YYYYMMDD} \
  --edt {结束日期YYYYMMDD} \
  --cash {初始资金} \
  --freq {F15/F30/F60}
```

**多频率对比回测**（用户要求对比不同频率时，依次运行）：
```bash
cd {baseDir}/../../ && python scripts/backtest_report.py --symbol {股票代码} --freq F30
cd {baseDir}/../../ && python scripts/backtest_report.py --symbol {股票代码} --freq F60
cd {baseDir}/../../ && python scripts/backtest_report.py --symbol {股票代码} --freq F15
```

**完整参数列表**：
- `--symbol`：股票代码（必填），沪市 `.SH`，深市 `.SZ`
- `--mode`：`dragon`（纯双龙）/ `fused`（缠论+双龙融合）/ `both`（两者对比，默认）
- `--sdt`：回测开始日期（YYYYMMDD），默认 365 天前
- `--edt`：回测结束日期（YYYYMMDD），默认今天
- `--cash`：初始资金（默认 100 万）
- `--freq`：K 线频率，可选 `F15` / `F30`（默认）/ `F60`

### Step 3：解读回测结果

回测报告必须包含以下核心指标，并给出通俗解读：

| 指标 | 含义 | 健康范围参考 |
|------|------|------------|
| 总收益率 | 策略在回测期内的总回报 | > 0% 即正收益 |
| 年化收益 | 折算为年度收益率 | > 5% 为良好 |
| 胜率 | 盈利交易占比 | > 40% 为良好 |
| 盈亏比 | 平均盈利 / 平均亏损 | > 1.5 为良好 |
| 最大回撤 | 从峰值到谷底的最大跌幅 | < -5% 为可控 |
| 夏普比率 | 风险调整后收益 | > 1.0 为优秀 |
| 交易次数 | 触发买卖的总次数 | 视波动而定 |
| 加仓次数 | 持仓中遇到更强信号加仓的次数 | — |

### Step 4：根据结果给出操作建议

| 评级 | 条件 | 建议 |
|------|------|------|
| 优秀 | 收益>5%, 胜率>50%, 盈亏比>2 | 可考虑纳入自选，正常仓位参与 |
| 及格 | 收益>0%, 胜率>35% | 轻仓参与，严格止损 |
| 不及格 | 收益<0% | 该策略不适合此股票，建议回避 |

附加判断维度：
- 若 **ADX < 20**：提示该股票趋势弱，策略天然不适配
- 若 **交易次数为 0**：提示信号条件严格，该时段未触发买入
- 若不同频率结果差异大：建议用户选择表现更好的频率级别

### Step 5：引导下一步

- 对比不同模式（融合 vs 纯双龙）
- 对比不同频率（F30 vs F60），验证策略稳健性
- 加入自选池（→ qstock-portfolio 技能）
- 查看缠论深度分析（→ qstock-advisor 技能）
- 回测其他股票

## 策略参数说明（v8）

当前回测使用 v8 策略参数：

```
频率级别：      默认 30 分钟（可切换 15/60 分钟）
dragon_lookback: 5（EMA/MACD 金叉共振窗口）
ATR 止损：      2.0 × ATR（下限 -3%, 上限 -8%）
移动止盈激活：   1.5 × ATR（后备 +5%）
移动止盈回撤：   1.0 × ATR（后备 3%）
时间止损：      40 根 K 线（F30 约 5 个交易日）
连亏暂停：      3 次
量能过滤：      开启（放量加仓 / 缩量减仓）
初始资金：      100 万
手续费：        万三（双向）
印花税：        千一（仅卖出）
滑点：          0.1%
T+1 约束：      严格执行
```

**不同频率下的时间对应关系**：

| 参数 | F15 | F30 | F60 |
|------|:---:|:---:|:---:|
| 每日 K 线数 | 16 | 8 | 4 |
| time_stop=40 对应天数 | 2.5 天 | 5 天 | 10 天 |
| lookback=5 对应时长 | 75 分钟 | 2.5 小时 | 5 小时 |

## 重要约束

- 历史回测不代表未来收益
- 回测结果受训练/验证集划分影响
- 不同市场环境（牛/熊/震荡）表现差异大
- 策略在 ADX < 20 的弱势股上表现较弱
- 高价股（如茅台）需要 100 万以上资金才能买入 1 手
