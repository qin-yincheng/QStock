# 双龙战法详解

## 战法原理

双龙战法利用两个独立的技术指标共振来确认趋势启动时机，减少单一指标的误判。

### 第一龙：EMA 均线系统

- **指标**：EMA5（5 周期指数移动平均线）和 EMA10（10 周期指数移动平均线）
- **金叉**：EMA5 从下方上穿 EMA10 → 短期趋势转多
- **死叉**：EMA5 从上方下穿 EMA10 → 短期趋势转空
- **多头排列**：EMA5 > EMA10 且均向上 → 趋势延续

**EMA 特点**：
- 比 SMA 更贴近价格，反应更灵敏
- EMA5 代表一周均价（5 根 K 线），EMA10 代表两周均价
- 金叉意味着短期均价开始超过中期均价，买方力量增强

### 第二龙：MACD 系统

- **指标**：DIF（快线）和 DEA（慢线），以及 MACD 柱状图
- **参数**：标准参数 (12, 26, 9)
- **金叉**：DIF 从下方上穿 DEA → 中期动能转多
- **死叉**：DIF 从上方下穿 DEA → 中期动能转空
- **零轴以上金叉**：DIF > 0 且 DEA > 0 时金叉 → 强势信号

**MACD 特点**：
- 反映的是中期趋势动能
- 零轴是多空分界线
- 柱状图递减 = 趋势动能减弱（背驰信号）
- 比 EMA 更滞后，但更可靠

### 双龙齐飞

**触发条件**：
```
在最近 5 根 K 线（lookback_window=5）窗口内：
  EMA5 上穿 EMA10（第一龙金叉）
  AND
  MACD DIF 上穿 DEA（第二龙金叉）
```

**信号强度分级**：

| 信号 | 条件 | 强度 |
|------|------|------|
| STRONG_BUY | 双龙齐飞 + DIF > 0（零轴以上） | 最强 |
| BUY | 双龙齐飞 + MACD 柱正（零轴附近） | 强 |
| BUY_EARLY | 仅 EMA 金叉 + MACD 方向向上（未完全金叉） | 早期 |
| SELL | EMA 死叉 或 MACD 死叉 | 卖出 |
| HOLD | 无明确信号 | 中性 |

### 辅助指标

双龙信号模块（v4）还计算以下辅助指标：

- **ATR14**：14 周期平均真实波幅，用于 ATR 自适应止损
- **vol_ratio**：当前成交量 / 20 周期成交量均线，用于量能过滤
  - vol_ratio > 1.2 → 放量，加大仓位
  - vol_ratio < 0.5 → 缩量，减小仓位

## 参数说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| lookback_window | 5 | EMA/MACD 金叉共振时间窗口（根K线数） |
| ema_fast | 5 | 快速 EMA 周期 |
| ema_slow | 10 | 慢速 EMA 周期 |
| macd_fast | 12 | MACD 快线周期 |
| macd_slow | 26 | MACD 慢线周期 |
| macd_signal | 9 | MACD 信号线周期 |
| require_macd_positive | False | 是否要求 MACD > 0（v3 起关闭，零轴附近金叉也有效） |

## 历史演进

| 版本 | 变更 |
|------|------|
| v1 | 基础 EMA + MACD 双金叉检测 |
| v2 | 新增 STRONG_BUY / BUY 分级 |
| v3 | lookback 3→5, 关闭 MACD>0 限制, 新增 BUY_EARLY |
| v4 | 新增 ATR14 和 vol_ratio 辅助指标 |

## 与缠论的配合

缠论负责"方向判断"，双龙负责"时机确认"：

```
缠论：当前处于二买回调后企稳 → 方向确认（可以做多）
双龙：EMA+MACD 刚好双金叉 → 时机确认（现在入场）
融合：BUY（75%）→ 建仓 30%
```

这种配合的核心逻辑是：
- 缠论的结构判断更准确但更滞后
- 双龙的动量信号更及时但容易被假突破误导
- 两者共振时准确率最高
