# v8 策略参数全量说明

## 当前推荐配置

```yaml
# === 基础参数（v7.2 继承） ===
dragon_lookback:        5       # EMA/MACD 金叉共振窗口（根K线数）
stop_loss:              -5%     # ATR 关闭时的固定止损
trailing_activate:      +5%     # ATR 关闭时的移动止盈激活阈值
trailing_stop_pct:      3%      # ATR 关闭时的移动止盈回撤幅度
time_stop_bars:         40      # 持仓超过 40 根 K 线无盈利则退出
max_consecutive_losses: 3       # 连续亏损 3 次后暂停交易

# === v8 新增 — ATR 自适应止损 ===
use_atr_stops:          True    # 开启 ATR 自适应止损
atr_stop_multiplier:    2.0     # 止损距离 = 2.0 × ATR / 买入价
atr_tp_activate:        1.5     # 移动止盈激活 = 1.5 × ATR
atr_trail_multiplier:   1.0     # 移动止盈回撤 = 1.0 × ATR
atr_stop_floor:         -3%     # 止损下限（最窄，波动小时兜底）
atr_stop_ceil:          -8%     # 止损上限（最宽，波动大时封顶）

# === v8 新增 — 量能过滤 ===
use_vol_filter:         True    # 开启量能仓位系数
# vol_ratio（当前量/20日均量）作为仓位乘数，范围 [0.3, 1.2]
# 放量（vol_ratio > 1）→ 加大仓位
# 缩量（vol_ratio < 1）→ 缩小仓位

# === 交易成本 ===
initial_cash:           1000000 # 初始资金 100 万
commission_rate:        0.0003  # 手续费万三（双向）
stamp_tax_rate:         0.001   # 印花税千一（仅卖出）
slippage:               0.001   # 滑点 0.1%

# === 仓位管理 ===
max_position_pct:       50%     # 单票最大仓位占总权益比
# 信号仓位分配：
#   STRONG_BUY: 40%
#   BUY:        30%
#   BUY_EARLY:  20%
#   WAIT_CONFIRM / BUY_WATCH: 15%
```

## 频率配置

| 参数 | F15 (15分钟) | F30 (30分钟) | F60 (60分钟) |
|------|:---:|:---:|:---:|
| bars_per_day | 16 | 8 | 4 |
| time_stop=40 对应天数 | 2.5 天 | 5 天 | 10 天 |
| lookback=5 对应时长 | 75 分钟 | 2.5 小时 | 5 小时 |
| 年化收益计算 | 242×16=3872 根/年 | 242×8=1936 根/年 | 242×4=968 根/年 |

**默认频率：30 分钟**（第九阶段结论）

## 九个阶段策略演进摘要

### 第一~四阶段：基础搭建

- 数据通路：Tushare + czsc ts_connector，修复 2 个 czsc bug
- 缠论引擎：czsc CZSC 对象，分型/笔/中枢/三类买卖点
- 双龙战法：EMA5/10 + MACD 金叉共振
- 回测引擎：T+1 / 手续费 / 印花税 / 滑点，首版参数

### 第五~六阶段：Agent 集成 + 选股优化

- OpenClaw Skill 开发，CLI 脚本（analyze / scan / backtest_report）
- 24 只股票 × 4 种策略批量对比 → ADX 分组选股规则
- 结论：底层策略收益不足是核心瓶颈

### 第七阶段：策略系统性优化

- 买入信号放宽（lookback 3→5，新增 BUY_EARLY）
- 缠论信号扩展（+4 新信号 +3 过滤器）
- 信号融合优化（AND→趋势确认，分层卖出）
- 退出机制优化（止损 -5%，移动止盈 +5%/-3%，时间止损 40K 线）
- 仓位管理（权益仓位 + 加仓机制 + max 50%）
- **宁德时代从 -9.32% → +5.61%，改善 14.93pp**

### 第八阶段：收益提升探索（6 组 A/B 实验）

| 实验 | 结论 |
|------|------|
| 8.1 日线趋势硬过滤 | ❌ 级联时间线偏移，有害 |
| 8.2 ATR 自适应止损 | ✅ 中性偏正 |
| **8.3a ATR + 量能过滤** | **✅ 最优，采纳为 v8** |
| 8.3b ATR + 弱信号缩减 | ❌ 有害 |
| 8.4a 8.3a + 冷却期 | ⚠️ 无帮助 |

**核心认知**：
- **硬过滤有害**（阻止信号 → 级联偏移 → 不可预测后果）
- **软过滤有效**（量能仓位系数 → 改善风险收益比 → 不破坏交易序列）
- **简洁组合优于复杂堆叠**

v8 vs v7.2 验证集数据：
- 逐股胜率：8/10
- 盈亏比提升：+34%（1.31 → 1.75）
- 最大回撤降低：-17%（-1.83% → -1.52%）

### 第九阶段：多频率扩展验证

**10 只股票全频率对比**：

| 指标 | 60min | 30min | 15min |
|------|:---:|:---:|:---:|
| 训练集平均收益 | -0.95% | **+0.47%** | -0.10% |
| **验证集平均收益** | +0.46% | **+1.22%** | -0.38% |
| 验证集胜率 | 28.0% | **36.5%** | 29.7% |

**结论**：30 分钟 > 60 分钟 > 15 分钟
- 30 分钟训练集从亏变盈，是八个阶段优化都未实现的突破
- 15 分钟噪声过大，高波动股（宁德/隆基/东财）信号崩塌
- v8 参数无需针对 30 分钟重新调参，直接适用

## 策略定位与局限

**定位**：趋势跟随型策略
- 适合 ADX > 20 的趋势股
- 不适合 ADX < 20 的弱势震荡股
- 高波动成长股受益最大（宁德时代、隆基绿能）
- 低波动蓝筹较弱（茅台、中国平安）

**已知局限**：
- 震荡市表现较弱
- 训练集整体仍为微亏（30min +0.47%），策略并非"稳赚不赔"
- 高价股需要大资金（茅台 1 手 ≈ 15 万）
- Tushare 数据延迟约 1 天，实时数据来自新浪（延迟约 15 分钟）
